<h1>PrepAI</h1>

<p>
  An AI-powered interview preparation platform built using ReactJS, FastAPI, LangChain, and Groq LLM.
</p>


<h2>Overview</h2>

<p>
PrepAI is a full-stack AI interview preparation platform designed to simulate realistic technical interview experiences. The platform generates personalized preparation roadmaps, dynamic interview questions, AI-driven feedback, and customized daily schedules based on the user's role, experience level, and tech stack.
</p>

<p>
The application combines modern frontend development with LLM-powered backend intelligence to create an adaptive interview preparation environment.
</p>


<h2>Features</h2>

<ul>
  <li>AI-generated interview preparation roadmap</li>
  <li>Dynamic technical interview questions</li>
  <li>AI-powered answer evaluation and feedback</li>
  <li>Personalized daily preparation schedule</li>
  <li>Multi-question interview workflow</li>
  <li>Real-time frontend and backend integration</li>
  <li>Modern responsive UI</li>
  <li>Structured backend architecture using FastAPI</li>
  <li>Prompt-engineered LLM workflows</li>
</ul>

<h2>Application Workflow</h2>

<ol>
  <li>User enters role, experience level, and tech stack</li>
  <li>AI generates a personalized interview roadmap</li>
  <li>AI creates a customized preparation schedule</li>
  <li>User launches AI interview session</li>
  <li>Dynamic interview questions are generated</li>
  <li>User submits answers</li>
  <li>AI evaluates responses and provides feedback</li>
  <li>User continues through multiple interview questions</li>
</ol>
